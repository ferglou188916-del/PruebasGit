Primera versión con Git

Git fue creado pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

Un sistema de control de versiones distribuido, conocido como DVCS, permite que cada desarrollador tenga una copia completa del historial del proyecto. Esto facilita el trabajo colaborativo, la creación de ramas y la sincronización de cambios entre repositorios locales y remotos.
